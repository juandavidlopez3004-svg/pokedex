## Paso a paso para crear la cuenta en Azure

### 1) Acceder al portal de alta de Azure
Ingrese al portal oficial de registro:

```bash
https://azure.microsoft.com/es-es/free
```

### 2) Iniciar el proceso gratuito
Hice clic en **Comenzar gratis** y continue con el flujo de registro.

### 3) Iniciar sesion con cuenta Microsoft
Inicie sesion con mi cuenta Microsoft para asociar la suscripcion.

### 4) Verificar identidad y metodo de pago
Como Azure for Students no me funciono, complete el alta con tarjeta Nequi para validacion.

### 5) Confirmar activacion
Despues de la validacion, Azure habilito el acceso a la suscripcion gratuita y sus creditos promocionales.

### 6) Entrar al portal de administracion
Accedi al portal para crear recursos:

```bash
https://portal.azure.com
```

### 7) Crear recurso base de hosting
Dentro del portal cree un recurso de tipo `Static Web App` para publicar la aplicacion Angular.

### 8) Configurar dominio personalizado
Despues de tener el sitio activo, configure el dominio personalizado `www.jimuax.app`.
